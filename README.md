# GASTROMIR — HoReCa Supply Solutions

Современная система снабжения для ресторанов, кафе и отелей. Закупки в один клик с автоматическим формированием накладных.

## 🚀 Функционал

- **Каталог товаров**: Полный перечень позиций с актуальными ценами, поиском и фильтрацией по категориям.
- **Умная корзина**: Сохранение выбранных товаров между сессиями (LocalStorage).
- **Генерация документов**: Автоматическое создание накладных в формате Word (.docx) для печати и отчетности.
- **Адаптивный дизайн**: Сайт оптимизирован для работы на десктопах, планшетах и смартфонах.
- **Брендирование**: Интерфейс выполнен в фирменных цветах компании.

## 🛠 Технологический стек

- **Vue 3**: Прогрессивный JavaScript-фреймворк.
- **Vite**: Быстрая сборка проекта.
- **Pinia**: Управление состоянием (корзина).
- **Vue Router**: Навигация по страницам.
- **Lucide Vue Next**: Набор современных иконок.
- **Docx**: Генерация документов Word на стороне клиента.
- **VueUse Motion**: Плавные анимации интерфейса.

## 📦 Установка и запуск

1. **Клонируйте репозиторий** (или скачайте файлы).
2. **Установите зависимости**:
   ```bash
   npm install
   ```
3. **Запустите сервер разработки**:
   ```bash
   npm run dev
   ```
4. **Сборка для продакшена**:
   ```bash
   npm run build
   ```

## 📂 Структура проекта

- `src/assets/`: Статические ресурсы (логотип, стили).
- `src/components/`: Многоразовые компоненты (Navbar, Footer, CartModal).
- `src/views/`: Основные страницы сайта.
- `src/stores/`: Логика корзины (Pinia).
- `src/data/`: База данных товаров (`products.json`).
